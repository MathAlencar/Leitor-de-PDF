Projeto criado com o objetivo de realizar a leitura de extratos bancários em formato de PDF's, onde ele extrai o corpo do documento de cada página e atráves da biblioteca Tabula, realizo a extração da tabela de informações de cada página, onde após esse processo, realizo toda a manipulação de dados usando bibliotecas como cvs e pandas, o objetivo final após a leitura de cada pdf é entregar um arquivo xlsx no formato desejado de dados, para assim ser rodado em um DashBoard no excel usando VBA para filtrar e manipular informações relevantes para a equipe de crédito realizar a sua análise.
